1. Navigate to this directory:
   ```bash
   cd banking_frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. The app will be run at [http://localhost:3000](http://localhost:3000)
